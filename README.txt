TOPSHOP VERSIE 2.0


HOE WERKT TOPSHOP:

--------

Alle bestanden (en variablen etc.) zijn in het Engels om verwarring te verkomen. Dit doe ik liever dan 
alles in het Nederlands te doen.

--------

Als je je wilt registreren, kun je dit doen. Dit word opgeslagen in de database webbackend.
Als je wilt inloggen, dan zou het kunnen dat je de pagina eerst moet refreshen. Daarna kom
je bij het startscherm waar je verder kan navigeren.

--------

Admin rechten kun je veranderen door naar 'client' te gaan (in de database
die 'webbackend' heet) en de 'role' te veranderen naar 1.
Als je geregistreerd bent, dan sta je standaard op client en dit is dan '0'.

--------

De zoekfunctie van de laptop shop en de phone shop werken allebei. Je kunt ook zoeken
naar het type processor, bijvoorbeeld Intel. Dan word er gezocht naar alle beschikbare laptops
die een processor hebben van Intel. Of bijvoorbeeld de camera resolutie van een mobiel.

--------

Je kunt de orders (in de navbar is dat 'rapport') bekijken door 1 van de opties te kiezen. Dan klik je
op de submit button en krijg je een schema met daarin de orders. Het zelfde geld voor de stock.

--------

De clients staan ook netjes in een schema verwerkt als je navigeert naar 'Clients'.

--------

De homepage heeft helaas verder nog geen content, behalve de login en een carousel (bootstrap).

--------

Ik heb technieken gebruikt die ook in het boek van MBO-ICT backend staan. Voordat ik 
met dit project begon, heb ik al een webshop gebouwd en heb veel kunnen gebruiken
in deze webshop. Heel veel dingen zijn veranderd zoals de stijl en ik heb extra features 
toegevoegd: clients, een phoneshop en de stock en orders van de phones.

--------

Note: Er zijn 2 register.php's, 1 in de register map en 1 in de php map. In de register map staat
de form en in de php map staat de PHP code om je te kunnen registreren.

--------

De pdf's kunnen genereren maar het is niet bij elke perfect. Maar voor de rest wel overzichtelijk 
waardoor het niet heel erg is.

--------

De database staat in het mapje sql.
